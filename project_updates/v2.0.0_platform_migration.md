## Tổng quan

Chuyển đổi dự án từ một công cụ dự đoán riêng lẻ (WC2026 Prediction Tool) thành **nền tảng đa giải đấu** (Football Prediction Tool) hỗ trợ cả Cup và League.

## Thay đổi chính

### 🏗️ Kiến trúc
- Tích hợp `react-router-dom` — thiết lập routing đa trang.
- Tách ứng dụng WC2026 gốc vào route riêng biệt `/competition/wc26`.
- Tạo Landing Page (`/`) với thiết kế minimalist, hiệu ứng động.
- Tạo Competition Hub (`/hub`) — trung tâm điều hướng giải đấu.

### 📄 Tài liệu & Cấu hình
- Đổi tên dự án: `wc26-prediction-tool` → `football-prediction-tool`.
- Phiên bản: `0.1.0` → `2.0.0`.
- Viết lại `README.md` theo tầm nhìn nền tảng.
- Chuẩn hóa `deploy.ps1` với quy tắc branch `platform/...`.
